Frequently Asked Questions

What if my bank PDF is an image or a scanned copy?
V-Import easily reads native digital PDFs. For highly complex scanned images, simply run the file through a free offline tool like PDF24 first to make the text readable, or paste the data into our Universal Excel format!
Do I need internet to use it?
No! V-Import is 100% Offline. Your client data never leaves your computer. Internet is only required one time for the initial software license activation.
Does it work with older versions of Tally?
Yes. V-Import is optimized for seamless Port 9000 integration with Tally Prime, but it also features a built-in Legacy mode to perfectly support older Tally ERP 9 environments.
